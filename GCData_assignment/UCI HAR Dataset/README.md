The analyisis in folder are experiments carried on 30 volunteers, with each person performing on of 6 activities (WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING) wearing a smartphone. Using its embedded accelerometer and gyroscope, we captured 3-axial linear acceleration and 3-axial angular velocity at a constant rate of 50Hz.

To get tidy dataset on averages of each measurment by subject and activity analysis was performed as in script run_analysis.R using raw dataset available at link bellow.

====

Repo contains:

- "README.md" : this file

- "CodeBook.md" : md file clearly descripting the analysis performed to get tidy data

- "tidy_data.txt" : txt file of aforementioned tidy dataset

- "run_analysis.R" : R script used to clean and prepare the aforementioned tidy dataset

- raw datasets

====

The raw datasets used can be found at next link:

- UCI HAR Dataset 
	http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones
